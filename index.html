Welcome to SPMsgBox.

The SpMsgBox plugin is useful for all Web Applications. you can customize what you like. u just want to include the jquery and Style sheet with images. it is easiest one for display a alert messages in your Web Applications.

You can Customize the dialog what you like!..
For More options see Demo [Here](http://www.prabhusiva.in/2014/12/jmsgbox-examples.html):
Simply add in your web page in any event..

Syntax:

For Show 

$.SPMsgBox.Show("Message","Title");

For ShowSuccess 

$.SPMsgBox.ShowSuccess("Message","Title");

For ShowError 

$.SPMsgBox.ShowError("Message","Title");

For ShowWarning 

$.SPMsgBox.ShowWarning("Message","Title");

For ShowInfo 

$.SPMsgBox.ShowInfo("Message","Title");

Options

Use $.SPMsgBox.options={}; for Setting all the Options for Your Page
Use Options in Layout page Or in Each Page for setting the properties what you like 

$.SPMsgBox.options = {
"autoHide": false,
"progressBar": true,
"hideCloseIcon": false,
"position": "top-right",
"stayTime": "3000",
"inEffectDuration": "600",
"bkColor1": "#1c95eb",
"pBarColor": "#b5f507",
"Msg": "You like this Dialog"
 };
